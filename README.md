# docker-liquibase
Docker image for running liquibase migrations

https://hub.docker.com/r/driveclutch/docker-liquibase/


### Environment Variables
- PGS_HOST
- PGS_PORT
- PGS_DB
- PGS_USERNAME
- PGS_PASSWORD
- SCHEMA
- SLACK_WEBHOOK
- LIQUIBASE_CONTEXTS
see [Contexts](https://www.liquibase.org/documentation/contexts.html)
